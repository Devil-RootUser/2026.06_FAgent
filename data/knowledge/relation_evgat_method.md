# Relation-EVGAT Method Notes

Relation-EVGAT uses node prediction error and edge degradation evidence for industrial time-series anomaly diagnosis. The edge vector compares relation components between normal and anomaly windows, such as correlation, absolute correlation, lag-like relation and direction-like relation. These relation changes are diagnostic evidence and should not be described as a strict causal chain.

Recommended report wording: "relation degradation evidence indicates that the linked sensors should be checked together." Avoid claiming that one sensor directly caused another sensor unless field evidence confirms it.
