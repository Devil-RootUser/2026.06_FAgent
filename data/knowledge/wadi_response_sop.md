# WaDI Alarm Response SOP

When a WaDI-like water treatment event is detected, first verify the top ranked sensor raw values, sampling quality and physical range. Then inspect upstream and downstream actuators, valves, pumps and control commands in the same event window. If relation degradation is high between two variables, compare their expected operating dependency with historical normal windows.

A practical checklist is: confirm sensor health, check actuator state, review control command history, compare upstream and downstream flow or level variables, and record operator notes for the event window.
